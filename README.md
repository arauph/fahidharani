## GitHub Pages Hosting Guide

### Step 1 — Create GitHub Repository
1. Go to https://github.com and sign in
2. Click "+" → "New repository"
3. Repository name: fahidharani
4. Set to Public
5. Click "Create repository"

### Step 2 — Upload Files
Drag ALL files from this folder into the repository upload page.
For the .github/workflows/deploy.yml file, create the folders manually.

### Step 3 — Enable GitHub Pages
1. Repository Settings → Pages
2. Source: "GitHub Actions"
3. Wait 2 minutes

### Step 4 — Live at:
https://fahidharani.github.io/fahidharani
